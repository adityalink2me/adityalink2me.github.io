The Image Production Pipeline

High quality images come from a clear production path.

1Goal

Decide what the image must do.

2Subject

Define the person, product, object, or scene.

3Style

Choose photo, 3D, illustration, graphic, or UI.

4Composition

Control angle, framing, spacing, and focus.

5Quality

Add lighting, detail level, ratio, and constraints.

6Repair

Fix only what is broken.

7Export

Save in the right size and format.

Rule: A prompt is a creative brief. The clearer the brief, the less random the image.

Different Types of Images

Choose the image family before choosing fancy words.

Realistic photo

Best for people, places, products, food, events, lifestyle campaigns, and believable scenes.

Product or studio render

Best for clean ecommerce visuals, packaging mockups, app devices, and object-focused hero images.

Illustration or character

Best for education, storytelling, kid-friendly content, mascots, explainers, and concept visuals.

Graphic or banner

Best for posters, social ads, website headers, thumbnails, event announcements, and offers.

Restoration or enhancement

Best for old photos, blurry images, low-resolution screenshots, damaged memories, and noisy scans.

Localized image edit

Best when the image is mostly right and only one object, background, color, pose, or expression should change.

Step 1 | Master Prompt
Copy-ready master promptAct as a senior AI image director, photographer, and graphic designer. Create one high-quality image for [PURPOSE] about [SUBJECT] for [AUDIENCE]. Image type: - [REALISTIC PHOTO / PRODUCT RENDER / CHARACTER ILLUSTRATION / SOCIAL MEDIA BANNER / POSTER / WEBSITE HERO / THUMBNAIL / RESTORATION / LOCALIZED EDIT] Visual direction: - Main subject: [WHO OR WHAT MUST APPEAR] - Setting/background: [LOCATION OR BACKDROP] - Style: [PHOTO / 3D / VECTOR / EDITORIAL / MINIMAL / PREMIUM / PLAYFUL / CINEMATIC] - Composition: [CLOSE-UP / WIDE / CENTERED / RULE OF THIRDS / LEFT TEXT SPACE / RIGHT TEXT SPACE] - Lighting: [SOFT NATURAL / STUDIO / DRAMATIC / BRIGHT CLEAN / GOLDEN HOUR] - Color palette: [BRAND COLORS OR MOOD] - Aspect ratio: [1:1 / 4:5 / 9:16 / 16:9 / A4 POSTER] - Quality target: sharp, clean, high resolution, polished, professional, no clutter. Reference instruction: - [NO REFERENCE / USE MY UPLOADED IMAGE FOR IDENTITY OR PRODUCT CONSISTENCY / USE MY LOGO ONLY AS BRAND COLOR REFERENCE] Strict rules: - Do not add random text, subtitles, watermarks, distorted logos, extra fingers, or messy details. - If text is needed, leave clean empty space and tell me where to add it manually. - Keep the image family-friendly and suitable for public use. - Preserve identity, product shape, and brand colors when a reference is provided. - Do not imitate a living artist or copyrighted character style. Before generating, silently check: subject, style, composition, lighting, color, aspect ratio, and what must not change. Generate the image now.

  2. Keep only the lines that matter.
  3. Paste into Gemini, ChatGPT, Canva, or another image tool.
  4. Generate 2-4 options.
  5. Use a repair prompt instead of starting over.

How Image Prompting Actually Works

The model builds a visual probability map.

It does not read your mind. It combines patterns from your words: subject, style, composition, lighting, context, and restrictions.

SubjectStyleCompositionLightingConstraints

Why prompts fail

  1. The subject is vague.
  2. The style words conflict.
  3. Too many ideas compete.
  4. The image needs readable text.
  5. The edit instruction changes the whole image.
  6. The reference instruction is unclear.
Better prompts reduce ambiguity one decision at a timeSlide 8
Explain that the prompt is a set of weighted instructions. Strong, specific instructions usually guide the output better.
The Prompt Formula
Reusable formula[IMAGE TYPE] of [MAIN SUBJECT], doing/appearing [ACTION OR STATE], in/against [SETTING OR BACKGROUND]. Style: [PHOTO / 3D / ILLUSTRATION / GRAPHIC / VECTOR / CINEMATIC / MINIMAL] Composition: [CAMERA ANGLE], [FRAMING], [WHERE THE SUBJECT SITS], [EMPTY SPACE FOR TEXT IF NEEDED] Quality: [LIGHTING], [COLOR PALETTE], [DETAIL LEVEL], [MOOD], [ASPECT RATIO] Reference: [WHAT TO PRESERVE FROM UPLOADED IMAGE] Do not: [TEXT, WATERMARKS, DISTORTIONS, EXTRA OBJECTS, STYLE CONFLICTS]

Why this format works

  • It separates creative decisions.
  • It makes edits easier.
  • It supports any image tool.
  • It leaves room for manual design text.
  • It avoids one long confusing sentence.
Use the same formula for photos, illustrations, products, and bannersSlide 9

Ready-to-use exampleRealistic premium studio photo of fresh chocolate cakes and colorful cupcakes arranged on a clean wooden bakery counter. Composition: Wide 16:9 website hero image, main cakes on the left third, clean empty warm background space on the right for a headline to be added later, eye-level camera, shallow depth of field. Using Uploaded Images as Reference
Character reference

Upload a character sheet or best previous image. Ask for the same face, outfit, colors, and proportions.

Product reference

Upload your product photo. Ask the tool to keep the product shape, color, and key details recognizable.

Brand reference

Upload logo or brand guide only if allowed. Ask for subtle colors or background presence, not heavy promotion.

Style reference

Use style references carefully. Avoid copying a living artist or copyrighted show style directly.

Scene reference

Upload a room, school, office, or background image. Ask to preserve layout, mood, or color only.

No reference

Ask for an original consistent design and reuse the best generated frame as the future reference.

References are for consistency, not blind copyingSlide 12
If learners use real people, remind them to get permission and avoid misleading contexts.
Reference Image Prompt
Copy-ready reference promptUse the uploaded image as a visual reference for [CHARACTER / OBJECT / PRODUCT / LOCATION]. Keep these consistent: - Overall shape and proportions. - Main colors. - Important visible details. - Mood and personality. - Outfit or surface design if relevant. Change these as needed for the new image: - Pose and action. - Camera angle. - Background. - Lighting. - Facial expression. Do not add subtitles, captions, labels, or readable text inside the image. Do not change the identity of the character/object/product. Create the image in this style: [ART STYLE]. Image to generate: [PASTE IMAGE DESCRIPTION]

Good reference habit

After generating one strong image, reuse it as a reference for future versions, matching products, character consistency, or brand style.

Image Restoration Workflow

Restoration works best when you ask for preservation first, enhancement second.

1Upload

Use the clearest original file.

2Preserve

Name identity, pose, clothing, and era.

3Repair

Fix scratches, blur, noise, fading, or tears.

4Enhance

Sharpen face, restore contrast, improve light.

5Respect

Do not invent new features.

6Upscale

Export higher resolution after repair.

7Compare

Check before and after closely.

Restoration rule: keep the person or object recognizable. A prettier face that changes identity is a failed restoration.

Preserve truth before improving qualitySlide 16
Use this for old family photos, scanned certificates, product shots, and blurry event images.
Copy-ready Restoration Prompt
Restore and enhanceRestore this uploaded image in high quality while preserving the original identity, pose, clothing, object shapes, and overall memory. Fix: - Blur, low resolution, compression noise, scratches, dust, stains, fading, and uneven lighting. Enhance: - Natural sharpness, realistic skin or surface texture, balanced contrast, clean background, and clear facial or product details. Do not change: - Face shape, age, expression, clothing design, product shape, body proportions, original era, or important objects. Do not add: - New people, decorative effects, fake text, heavy beauty filters, unrealistic colors, watermark, or artificial-looking skin. Output: A natural high-resolution restored version that still looks like the same original image.

Restoration checks

  1. Does the face or product still match?
  2. Did it invent details?
  3. Are eyes, hands, and edges natural?
  4. Is the image too smooth?
  5. Is the final size useful?
Restoration prompts should say what must not changeSlide 17
Tell learners to compare side-by-side, not from memory.
High-quality Upscale and Cleanup
Upscale promptUpscale and enhance this image for professional use. Target quality: - High resolution, crisp subject edges, clean texture, realistic detail, balanced contrast, accurate colors, and no compression artifacts. Preserve: - Original composition, identity, product shape, clothing, background layout, logo placement if present, and overall mood. Fix only: - Noise, pixelation, blur, dull colors, small scratches, harsh shadows, and low contrast. Do not: - Change the face, product design, text, logo, pose, camera angle, background objects, or overall style. If any readable text or logo is unclear, do not invent it. Keep it blank or preserve the original as closely as possible.

Making Graphics and Banners from Generated Images

Best workflow

  1. Generate the background or hero visual without text.
  2. Leave clean space for headline and logo.
  3. Open the image in Canva, Figma, PowerPoint, or Adobe Express.
  4. Add real text, logo, buttons, and shapes manually.
  5. Check mobile crop and readability.
  6. Export the correct size.

Why not ask AI for the final banner?

Image tools often make bad spelling, distorted logos, and unreadable layout. Use AI for the visual foundation; use a design editor for exact text and branding.

How to Create High-quality Graphics Nicely
One focal point

Do not make every object important. The viewer should know where to look first.

Clean negative space

Leave calm areas for headline, logo, product name, or call-to-action.

Controlled palette

Use 2-4 main colors with enough contrast. Match brand colors when needed.

Real typography later

Add readable text manually in a design editor. Do not trust generated text.

Right crop

Generate or edit for the actual platform ratio before final export.

Final inspection

Zoom in for hands, faces, product details, logos, spelling, and edge artifacts.

Polish comes from composition and restraintSlide 22
Use this as a final checklist before participants submit assets.
Edit One Part Without Affecting the Whole Image
Localized edit promptEdit only the selected area: [OBJECT / BACKGROUND PART / CLOTHING / COLOR / EXPRESSION / LIGHTING]. Change: [EXACT CHANGE YOU WANT] Keep everything else unchanged: - Same person or product identity. - Same pose, camera angle, composition, background layout, lighting direction, colors, and image style. - Same facial features, clothing shape, object proportions, and surrounding details. Do not regenerate the whole image. Do not add new objects, text, watermark, logo, people, hands, or decoration. Blend the edit naturally so it matches the original image quality and lighting.

Specialized examples

  • Change only shirt color.
  • Remove only background clutter.
  • Replace only sky or wall.
  • Brighten only face or product.
  • Make only one object larger.
  • Keep the rest pixel-for-pixel similar.
