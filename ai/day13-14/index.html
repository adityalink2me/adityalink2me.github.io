How the Combined Day Works

Day 13: Image production

  1. Choose a video idea.
  2. Write the short script.
  3. Break it into scenes.
  4. Define the visual style.
  5. Create consistent image prompts.
  6. Generate and download scene images.

Day 14: Video generation

  1. Open Flow or another video tool.
  2. Upload scene images as start/end frames.
  3. Paste the scene description, animation, and voiceover.
  4. Generate short clips.
  5. Review, repair, and export.
  6. Combine clips into one story.

The Production Pipeline

Every good AI video starts before the video tool opens.

1Idea

Pick one clear topic or story.

2Script

3Scenes

6Video

7Edit

Video type

Explainer, story, product demo, awareness message, lesson, reel, or ad concept.

Audience

Kids, parents, students, customers, teachers, founders, or general public.

Language

Nepali, English, Hindi, bilingual, or another language.

Visual style

2D cartoon, close-to-human, fruit characters, superhero, 3D toy, paper cutout.

Reference

Own image, character sheet, brand colors, product photo, logo, or no reference.

Brand presence

Visual Style Menu

Give learners style choices instead of one fixed example.

Clean 2D cartoon explainer

Friendly characters, simple shapes, readable scenes, soft colors. Best for education and social media.

Close-to-human animated style

Realistic proportions, expressive face, modern clothes. Best for relatable workplace or life stories.

Fruit or object characters

Objects act like characters. Best for fun reels, food brands, kids' content, and metaphor stories.

Superhero or monster energy

Bold poses, transformation, exaggerated power. Best for hooks, challenges, and dramatic comparison videos.

3D toy or clay style

Miniature characters, tactile lighting, playful movement. Best for product explainers and kid-friendly shorts.

Minimal infographic animation

Icons, simple scenes, abstract objects. Best when people are not needed and clarity matters most.

Style is not decoration; it controls consistencySlide 5
Ask participants which style fits their own topic and why.
Step 1 | Master Prompt
Copy-ready master promptAct as a professional short-form visual storytelling director and AI production planner. Create a short [LANGUAGE] video about [TOPIC] for [AUDIENCE]. The video type is [VIDEO TYPE], such as educational explainer, product demo, awareness story, classroom lesson, brand reel, comedy scene, or mini story. Visual direction: - Art style: [ART STYLE] - Character type: [HUMAN / CLOSE-TO-HUMAN / FRUIT OR OBJECT CHARACTER / SUPERHERO / 3D TOY / MINIMAL INFOGRAPHIC] - Color palette: [COLOR PALETTE] - Setting style: [MODERN OFFICE / HOME / CAFE / SCHOOL / CITY / FANTASY / PRODUCT STUDIO] - Reference image use: [NO REFERENCE / USE MY UPLOADED IMAGE FOR CHARACTER CONSISTENCY / USE MY UPLOADED PRODUCT IMAGE / USE MY BRAND COLORS] - Brand presence: [NONE / SUBTLE BACKGROUND / SMALL WATERMARK / END-CARD MENTION] Strict rules: - Keep the output simple, visual, and easy to understand. - Do not sound like a sales pitch unless I explicitly ask for an ad. - Do not add subtitles, captions, labels, logos, or readable text inside generated images or video frames. AI often creates poor text. Leave clean space if text will be added manually later. - Maintain the same main character, object, or product style across all scenes. - Do not use internal body visuals unless the topic requires it and I approve. - Keep the content family-friendly. Video structure: - Exactly [SCENE COUNT] scenes. - Each scene should last [DURATION PER SCENE] seconds. - Total video length should be [TOTAL LENGTH]. - Fast-paced, clear, and suitable for [PLATFORM]. Step 1: Generate 10 curiosity-driven video ideas for this topic. The ideas must be visual, relatable, short-form friendly, and not too technical. After listing the ideas, stop and ask me which idea I want to expand. After I choose an idea, continue with: 1. Character or object design summary. 2. Exactly [SCENE COUNT] scene production cards. 3. Image generation prompt for each scene. 4. Animation prompt for each scene. 5. Voiceover script for each scene. Use this exact format for every scene: Scene 1 Production Card Scene Description: Image Prompt: Animation: VoiceOverScript: Reference Notes: Repeat the same format for every scene.

Step 2 | Write the Voiceover Before Images

Why script first?

Images should serve the voiceover. If the script is unclear, the visuals become random.

HookProblemSimple explanationExampleTakeaway
Script promptWrite a short [LANGUAGE] voiceover script for a [TOTAL LENGTH] video about [TOPIC] for [AUDIENCE]. Rules: - Use simple spoken language. - Start with a curiosity hook. - Explain one idea only. - Avoid heavy jargon. - Keep each scene line short enough for [DURATION PER SCENE] seconds. - Make the final line useful and natural. - Do not make it sound promotional unless the video is explicitly an ad. Output exactly [SCENE COUNT] voiceover lines, labeled Scene 1 to Scene [SCENE COUNT].

Step 3 | Production Card Format
Scene card formatUse this format for every scene so I can copy it into image and video tools easily. Scene [NUMBER] Production Card Scene Description: [Describe what we see. Include character action, setting, emotion, and key visual metaphor.] Image Prompt: [A clean image-generation prompt for this exact frame. Include art style, character consistency, environment, lighting, composition, and no text inside the image.] Animation: [Describe 6-8 seconds of motion. Include character movement, object movement, camera movement, and mood. Keep motion smooth and minimal.] VoiceOverScript: [One short spoken line for this scene in the chosen language.] Reference Notes: [Mention uploaded image, character sheet, product photo, brand colors, or "no reference".]

Why this format works

  • Scene Description helps planning.
  • Image Prompt helps Gemini or image tools.
  • Animation helps Flow or video tools.
  • VoiceOverScript keeps timing clear.
  • Reference Notes protect consistency.

Step 4 | Image Generation Prompt Template
Copy-ready image prompt templateCreate one image for Scene [NUMBER] of a short video. Scene: [PASTE SCENE DESCRIPTION] Voiceover context: [PASTE VOICEOVER SCRIPT] Visual style: [PASTE ART STYLE, CHARACTER TYPE, COLOR PALETTE, AND SETTING] Character consistency: Use the same main character/object/product as previous scenes. Keep face, outfit, colors, proportions, and overall design consistent. Reference image instruction: [IF UPLOADED: Use the uploaded image as a reference for character/object/product consistency. Do not copy copyrighted style unless I own or have permission to use it.] [IF NOT UPLOADED: No reference image. Create a consistent original design.] Image rules: - Show only one clear moment from the scene. - Make it look like a clean animated frame. - No subtitles, captions, labels, UI text, or readable words inside the image. - Leave clean space if text will be added manually later. - Use simple composition and clear emotion. - Family-friendly. Output only the final image.

Using Uploaded Images as Reference
Character reference

Upload a character sheet or best previous image. Ask for the same face, outfit, colors, and proportions.

Product reference

Upload your product photo. Ask the tool to keep the product shape, color, and key details recognizable.

Brand reference

Upload logo or brand guide only if allowed. Ask for subtle colors or background presence, not heavy promotion.

Style reference

Use style references carefully. Avoid copying a living artist or copyrighted show style directly.

Scene reference

Upload a room, school, office, or background image. Ask to preserve layout, mood, or color only.

No reference

Ask for an original consistent design and reuse the best generated frame as the future reference.

References are for consistency, not blind copyingSlide 12
If learners use real people, remind them to get permission and avoid misleading contexts.
Reference Image Prompt
Copy-ready reference promptUse the uploaded image as a visual reference for [CHARACTER / OBJECT / PRODUCT / LOCATION]. Keep these consistent: - Overall shape and proportions. - Main colors. - Important visible details. - Mood and personality. - Outfit or surface design if relevant. Change these as needed for the new scene: - Pose and action. - Camera angle. - Background. - Lighting. - Facial expression. Do not add subtitles, captions, labels, or readable text inside the image. Do not change the identity of the character/object/product. Create the image in this style: [ART STYLE]. Scene to generate: [PASTE SCENE DESCRIPTION]

Good reference habit

After generating Scene 1, use the best Scene 1 image as the reference for Scene 2. This usually improves consistency.

Reference images make scene-to-scene continuity easierSlide 13

Repair Bad Images Fast
Image repair promptRegenerate this image with the following fixes while keeping the same character, style, scene, and composition: Problems to fix: - [PROBLEM 1] - [PROBLEM 2] - [PROBLEM 3] Keep: - [WHAT IS WORKING] Change: - [WHAT MUST CHANGE] Important rules: - No subtitles, captions, labels, logos, watermarks, or readable text inside the image. - Keep hands, face, eyes, and proportions clean. - Keep the main character/object consistent with the reference. - Make the image simple, clear, and usable as a video start frame.

Start Frame and End Frame Choices
Start frame only

Use when the scene has simple movement and you want the tool to decide the ending.

Start and end frame

Use when the character must move from one clear pose or position to another.

Same image twice

Use when you want subtle camera motion, blinking, breathing, or gentle object movement.

Different angle end frame

Use carefully. Big angle changes can cause identity drift or strange transitions.

Product scene

Use stable frames. Ask for slow rotation, hand movement, light sweep, or camera push.

Character scene

Keep movement small: gesture, turn head, point, walk a few steps, react.

The more controlled the frames, the easier the clip is to useSlide 18
Explain that huge movement usually increases errors.
Animation Prompt Patterns
Explainer
6-8 seconds. Slow camera push-in. The character points at the laptop, nods, and a simple visual metaphor appears beside them. Keep motion smooth and minimal.
Product
6-8 seconds. The camera slowly circles the product by a few degrees. Soft light moves across the surface. Keep product shape and logo unchanged. No text.
Fruit character
6-8 seconds. The fruit character bounces gently, looks surprised, then smiles. Background objects move slightly. Keep face and body design consistent.
Superhero
6-8 seconds. The character takes a confident step forward, cape moves gently, camera tilts upward slightly, energy glow appears in the background. No readable text.
Close-to-human
6-8 seconds. The character looks at the phone, reacts with a small smile, and turns toward camera. Natural movement, no face distortion, no extra people.
Infographic
6-8 seconds. Icons move from left to right in a simple sequence. Camera remains static. Use clean motion and no text labels.
Motion prompts should be short, clear, and controlledSlide 19
Learners can copy one pattern and adapt it to their scene.
Repair Bad Video Clips
Video repair promptRegenerate the clip using the same uploaded image and same scene idea. Fix these problems: - [PROBLEM 1] - [PROBLEM 2] - [PROBLEM 3] Keep these parts: - [WHAT WORKED WELL] New motion instruction: [WRITE A SIMPLER, MORE CONTROLLED MOTION] Strict rules: - Keep the character/object/product consistent with the uploaded frame. - Keep camera movement simple. - Do not add subtitles, captions, labels, watermarks, logos, or readable text. - Avoid face distortion, extra fingers, melting objects, sudden scene changes, and random extra characters. - Make the clip usable as one scene in a short video.

If the clip keeps failing

  1. Reduce movement.
  2. Use same start and end frame.
  3. Regenerate the image with cleaner composition.
  4. Remove extra characters.
  5. Shorten the animation instruction.
Most repairs are solved by simpler motionSlide 20
Do not let learners spend too long fixing one clip. Generate alternatives and pick the best.
Voiceover and Timing

Timing rule

A 6-8 second scene needs one short spoken line. If the line is too long, either split the scene or shorten the words.

One thoughtNatural speechNo jargonRead aloud
Timing repair promptShorten this voiceover so it fits naturally in [DURATION] seconds. Original line: [PASTE LINE] Rules: - Keep the meaning. - Use simple spoken [LANGUAGE]. - Make it sound natural, not robotic. - Avoid heavy jargon. - Return 3 options. - Mark the best option.
Shorter voiceover usually makes better videoSlide 21
Have learners read each line with a timer.
Step 7 | Assemble the Final Video

Minimum editing steps

  1. Import all clips in scene order.
  2. Trim bad starts or endings.
  3. Add voiceover or generated audio.
  4. Add subtitles manually if needed.
  5. Add background music quietly.
  6. Export vertical video if posting to short-form platforms.

Final quality check

  1. Does every scene match the script?
  2. Is the character consistent?
  3. Is there any unwanted text?
  4. Is the voiceover clear?
  5. Is the pacing fast enough?
  6. Does the ending feel complete?
